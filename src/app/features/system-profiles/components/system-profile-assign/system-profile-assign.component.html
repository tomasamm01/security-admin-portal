Información de Asignación
¿Cómo funciona?

Seleccione un sistema y luego elija los perfiles que desea asignarle. Un sistema puede tener múltiples perfiles asignados.

Tipos de Perfiles
  • Administrador: Acceso completo
  • Gestor: Gestión de usuarios
  • Operador: Operaciones básicas
  • Analista: Solo lectura y reportes
Consejo

Asigne solo los perfiles necesarios para cada sistema. Demasiados perfiles pueden complicar la gestión de permisos.

Seguridad

Las asignaciones determinan qué acciones pueden realizar los usuarios en cada sistema. Revíselas cuidadosamente.
